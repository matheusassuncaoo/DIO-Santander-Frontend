HTML, ou HTML, é a linguagem padrão para criar páginas web. É uma linguagem essencial para desenvolvedores. Aqui, usamos tags para estruturar conteúdo, como este texto sublinhado ou este em itálico. Antes, usávamos fontes coloridas, mas hoje o CSS é mais comum.

Passos para Criar uma Página HTML

  1. Definir o DOCTYPE
  2. Criar a tag <html>
  3. Adicionar <head> e <body>
